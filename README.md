## 🚀 Projetos em Destaque

### SGMI - Sistema de Gerenciamento de Manutenção Industrial
*Sistema completo para digitalização de manutenção e controle de ativos com integração de IA.*
- **Tech:** Python (Flask), MySQL, API Google Gemini.
- **Destaque:** Arquitetura 100% modular, RBAC, modelos 3D dinâmicos e assistente virtual integrado.
- 🔗 [Ver Repositório](https://github.com/icarodev10/SGMI)

### Frota Control
*Plataforma para gestão empresarial de frota de veículos.*
- **Tech:** FastAPI, SQLite, Render, Vercel.
- **Destaque:** Foco em construção de API robusta, deploy automatizado separando frontend e backend.
- 🔗 [Ver Repositório](https://github.com/icarodev10/frota_control)

### FaceLogger
*Sistema de controle de acesso simulando catracas via reconhecimento facial.*
- **Tech:** FastAPI, OpenCV, Arduino.
- **Destaque:** Integração completa entre o modelo de detecção facial, banco de dados e acionamento físico de hardware via comunicação serial com Arduino.
- 🔗 [Ver Repositório](https://github.com/icarodev10/FaceLogger)

### SafeVision
*Monitoramento de EPI (Equipamento de Proteção Individual) com IA.*
- **Tech:** YOLOv8, OpenCV, FastAPI.
- **Destaque:** Validação relacional em tempo real e geração automática de logs de auditoria de segurança.
- 🔗 [Ver Repositório](https://github.com/icarodev10/SafeVision-Showcase)
